# Caesar Cipher

## Description
This project implements the Caesar Cipher algorithm using Python. It allows users to encrypt and decrypt messages using a user-defined shift value.

## Features
- Encrypt text
- Decrypt text
- User-defined shift value
- Supports uppercase and lowercase letters
- Preserves spaces and special characters

## Technologies Used
- Python 3
